## 调权依据

- 高 `win_rate` + 高 `avg_return` → 强势信号, 应增加权重
- 低 `win_rate` 或负 `avg_return` → 弱势信号, 应降低权重

- `net_attribution` < 0 → 净拖累, 建议大幅降权或停用
